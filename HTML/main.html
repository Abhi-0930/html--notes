Semantic Tags in HTML

They help browsers and other software understand the structure of the document, which can be beneficial for accessibility, search engine optimization (SEO), and overall organization of the content
Some of them are
Header Tag
Defines a header for a document or section.
Nav Tag
Defines a set of navigation links.
Main Tag
Defines the main content of a document.
Article Tag
Defines an independent piece of content, such as a blog post or article
Section Tag
Defines a section of a document.
Footer Tag
Defines a footer for a document or section.
